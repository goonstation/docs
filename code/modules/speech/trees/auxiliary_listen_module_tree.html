/datum/listen_module_tree/auxiliaryAuxiliary listen module tree datums handle adding and removing their own modules to a specified target listen module tree, and transferring modules when the target changes. These are used as listen module trees for datums that frequently change between atoms with their own trees, such as clients or minds.
