/datum/listen_module/inputListen input module datums modify say message datums passed to them from the say channel that they are subscribed to, then in turn pass the message to their parent listen module tree for further processing.
/datum/listen_module/input/bundledBundled listen input module datums operate in a similar manner to ordinary listen inputs, however instead of receiving every message passed over an ordinary say channel, they are subscribed to specific subchannel of a bundled say channel.
