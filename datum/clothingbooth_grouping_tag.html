/datum/clothingbooth_grouping_tag

Tags can be used to further sort clothingbooth_groupings into more specific categorisations, such as by seasonality, formality, as a set that is intended to be paired with items of another grouping, etc. An arbitrary number of these can be assigned to any given grouping to an extent that is reasonable.

Vars

display_orderThe higher the display_order, the lower the clothingbooth_grouping_tags will be displayed (i.e., further right). Exists for the sake of a clear organizational hirearchy. Seasonal tags will be shown first, followed by formality, then set name, etc.
