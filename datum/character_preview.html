Character Preview
/datum/character_preview

Essentially, it creates a human on a 1x1 map and lets you set the appearance of that human.

This parent type is for use in single-client windows. See datum/character_preview/window for a detatched window and and datum/character_preview/multiclient for a multi-client variant.

Use winset() to position the control within the window.

See the default code for an example - places it at 0,0 and makes it 128x128 pixels.

Vars

preview_idThe map ID for use with winset().
preview_mobThe human mob shown in the preview. May be useful to access directly if you want to put clothes on it or whatever.

Procs
