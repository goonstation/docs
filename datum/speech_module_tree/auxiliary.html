Goonstation 13 - Modules - Types — Var Details

/datum/speech_module_tree/auxiliary

Auxiliary speech module tree datums handle adding and removing their own modules to a specified target speech module tree, and transferring modules when the target changes. These are used as speech module trees for datums that frequently change between atoms with their own trees, such as clients or minds.

Vars

target_speech_treeThe speech module tree that this auxiliary speech module tree should add and remove its modules to and from.
