/image/maptext

Maptext images are special subtypes of images without an icon with the sole purpose of displaying maptext to a single client. Maptext images should never be instantiated directly using new; use the NEW_MAPTEXT macro instead.

Vars

respect_maptext_preferencesWhether this maptext image should respect the client's flying chat preferences.

Procs

initThe parameters to /image/New() are defined internally by BYOND, which means that image subtypes cannot define their own New() parameters. This necessitates an init() proc if arguments are to be passed during instantiation.
